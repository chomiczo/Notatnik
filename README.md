# Prosty Notatnik w C# Windows Forms

To jest prosty notatnik napisany w C# przy użyciu Windows Forms. Projekt ten umożliwia użytkownikowi otwieranie, zapisywanie i edytowanie plików tekstowych, a także dostarcza funkcje zmiany koloru tła i czcionki.

## Funkcje

- **Otwieranie pliku:** Użytkownik może otworzyć istniejący plik tekstowy i edytować jego zawartość.

- **Zapisywanie pliku:** Projekt umożliwia użytkownikowi zapisanie aktualnej zawartości do nowego pliku tekstowego.

- **Zmiana koloru tła:** Użytkownik może zmieniać kolor tła obszaru edycji tekstu.

- **Zmiana czcionki:** Dostępna jest funkcja zmiany czcionki i koloru tekstu w obszarze edycji.

- **Skróty klawiszowe:** Skróty klawiszowe ułatwiają korzystanie z niektórych funkcji:
  - `Ctrl + A`: Zaznacz wszystko
  - `Ctrl + C`: Kopiuj zaznaczony tekst
  - `Ctrl + X`: Wytnij zaznaczony tekst
  - `Ctrl + V`: Wklej skopiowany tekst
  - `Ctrl + Z`: Cofnij ostatnią operację

## Jak uruchomić

1. Sklonuj repozytorium na swój lokalny komputer.
2. Otwórz projekt w Visual Studio (lub innym środowisku obsługującym C#).
3. Skompiluj i uruchom aplikację.

## Zalecenia

- Upewnij się, że masz zainstalowanego .NET Framework, aby uruchomić aplikację.
- Spróbuj dostosować i rozwijać kod zgodnie z własnymi potrzebami i pomysłami.
